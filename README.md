# Gym-Helps (ajudas para academia)💪🏋️

## Para que o site foi desenvolvido

- Este site foi desenvolvido para talvez ajudar a dar um ponta-pé inicial no mundo da academia, e tem funcionalidades como:

## 1. calculadora de Calorias
Calculadora de Calorias, ou, uma calculadora de gasto metabolico basal (TMB) e TDEE, que pode auxiliar com contas simples feitas com as medidas do corpo e taxa de atividade diária.

## 2. Calculadora de IMC
Calculadora com umas das mais famosas formulas deste mundo, que é a formula do IMC (índice de massa muscular). De um jeito simples mostra o número e traz consigo a tabela de IMC.

## 3. Exercícios e Músculos

A página que traz os músculos (Bíceps braquial, peito, tríceps, dorsal, ombro, abdômen, quadríceps, bíceps femoral, panturrilha e glúteo), e alguns exercícios bons e comuns para cada muscúlo (visando a hipertrofia principalmente).

## 4. Rotina de treinos

a ultima página foi feita de forma simples para ajudar a montar uma rotina de treino, adicionando exercícios, séries, cargas e repetições para diferentes dias e treinos e se organizar melhor entre dia de treino e descanso, além de computar cargas para melhor progressão.

Sobre TMB e TDEE

O TMB(Taxa Metabólica Basal), é a quantidade de calorias necessárias para manter órgãos, respiração e circulação funcionais em repouso. existem alguns modos para calcular este gasto, mas o usado para esse site é a fórmula comum de Mifflin-St Jeor, que se dá por:

TMB masculino = (10 x p) + (6.25 x a) - (5 x i) + 5

TMB feminino = (10 x p) + (6.25 x a) - (5 x i) - 161

sendo:
p = peso em kilogramas
a = altura em centimetros
i = idade em anos

Já o TDEE(Gasto Energético Diário Total) é o seu gasto metabólico basal multiplicado por um fator de atividade física, os valores para cada nivel de atividade são:

Sedentário = 1.2

Levemente Ativo = 1.375

Moderadamente Ativo = 1.55

Muito Ativo = 1.725

Extremamente ativo (normalmente atletas de alto rendimento) = 1.9

Com o valor de calorias, caso quiser fazer um superávit calorico(bulking), é só adicionar algumas calorias, caso contrário e quiser fazer um déficit calórico (cutting) é só retirar algumas calorias, e não esquecer dos treinos!
